# FitSystem - Complete Fitness Transformation App

A Progressive Web App (PWA) with personalized workout routines and meal plans for both male and female users.

## Features

- 6-day Push/Pull/Legs split for the male plan (lean bulk: 65 to 70 kg)
- 5-day Tone & Sculpt split for the female plan (fat loss + toning)
- Comprehensive exercise library with 6 variations each
- Individual YouTube form-video links for every variation
- Detailed Nepali and Western meal plans with full recipes
- Installable as a native-like app on iOS, Android, and desktop
- Works offline once installed

## Setup on GitHub Pages

1. Create a new repository on GitHub (or use an existing one)
2. Upload all files in this folder to the repository
3. Go to repository Settings -> Pages
4. Under Source, select "Deploy from a branch"
5. Select main branch and / (root), then Save
6. Wait 1-2 minutes for the site to build
7. Visit `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

## Installation as App

### iPhone / iPad (Safari)
1. Open the GitHub Pages URL in Safari
2. Tap the Share button (square with up arrow)
3. Scroll down and tap "Add to Home Screen"
4. Tap "Add"

### Android (Chrome)
1. Open the GitHub Pages URL in Chrome
2. Tap the three-dot menu
3. Tap "Add to Home Screen" or "Install App"
4. Tap "Install"

### Desktop (Chrome / Edge)
1. Open the URL
2. Click the install icon in the address bar (right side)
3. Click "Install"

## File Structure

- `index.html` - Profile selection screen
- `male.html` - Male training and nutrition plan
- `female.html` - Female training and nutrition plan
- `manifest.json` - PWA manifest
- `sw.js` - Service worker for offline support
- `icon-192.png`, `icon-512.png` - App icons
